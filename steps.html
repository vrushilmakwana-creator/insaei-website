1
Phase One Set Up Your Odoo Account & Website
  • 1.

    Go to odoo.com and sign up / log in

    Visit odoo.com → click Start for Free or Sign In. If new, choose the free plan. You will land on your Odoo dashboard.

  • 2.

    Activate the Website app

    From the main menu, go to Apps → search for Website → click Activate. This is free and included in every Odoo plan.

  • 3.

    Open the Website Editor

    Click the Website icon on your dashboard, then click Go to Website. You'll land on your Odoo website editor. You can now start adding content.

2
Phase Two Upload Static Files (CSS, JS, Images)
  • 1.

    Open Attachments Manager

    In Odoo backend, go to Website → Configuration → Attachments. This is where you upload static files.

  • 2.

    Upload all static assets

    Click Upload and select these files one by one. Once uploaded, click on each file to copy its public URL — you will need these URLs for the next phase.

    styles.css site.js photo1.jpg photo2.jpg durgesh.png
  • 3.

    Note the public URL pattern

    Odoo usually serves uploaded files at a URL like:
    /web/image/ir.attachment/[ID]/datas/filename.ext
    or sometimes directly at /web/content/[ID]?filename=styles.css.
    Write these URLs down — you'll replace the href="styles.css" and src="site.js" references in each HTML page.

Important: In each HTML file, the <head> has href="styles.css" and <body> has src="site.js". Before pasting the HTML into Odoo, replace these with the full Odoo attachment URLs you noted above. Same for photo1.jpg and photo2.jpg.
3
Phase Three Create Pages in Odoo Website Builder
  • 1.

    Create a new page in Odoo

    Go to your Odoo website editor → click Pages (top bar) → New Page. Give it the right name (e.g., "Home", "About", "Services", etc.).

  • 2.

    Switch to HTML / Code view

    In the page editor, click the </> or Code button in the Odoo toolbar (usually top-right). This opens the HTML editor.

  • 3.

    Paste your HTML page content

    Open your .html file in a text editor (Notepad, VSCode, etc.). Select all (Ctrl+A) → Copy → Paste into Odoo's HTML editor. Then click Save.

  • 4.

    Repeat for every page

    Do this for all 7 pages: index.html (set as Homepage), about.html, services.html, recruitment.html, marketing.html, careers.html, and contact.html.

  • 5.

    Set index.html as the Homepage

    Go to Pages → find your "Home" page → click the ⋮ (three dots) → Set as Homepage. This ensures your domain root (/) points to your main page.

4
Phase Four — Critical Setting Up the Contact Form in Odoo
How it works: Your contact form now submits to Odoo's own endpoint /website/form/crm.lead. This creates a CRM Lead directly inside Odoo every time someone fills in the form — no PHP needed. You can then view, reply, and manage enquiries from the Odoo CRM module. The form's design/styling remains 100% yours.
  • 1.

    Activate the CRM App in Odoo

    Go to Apps → search CRM → click Activate. This is required so Odoo knows where to store form submissions. Free plan includes CRM.

  • 2.

    Make sure the Website Form module is enabled

    In Odoo, go to Website → Configuration → Settings. Scroll to the Privacy section. Ensure Use Custom Privacy Policy is off (or set as needed). The form submission feature is enabled by default when CRM is installed.

  • 3.

    Test the contact form after going live

    Once your contact.html page is live on Odoo, fill in the form and submit it. Then go to CRM → Leads / Pipeline — you should see a new lead created with all the submitted details.

  • 4.

    View & reply to messages via Odoo CRM

    Click any lead → use the Chatter (message box at the bottom of the lead page) to send replies. You can also set up automated email notifications via CRM → Configuration → Stages.

  • 5.

    If the form doesn't work (fallback)

    Some Odoo free plans restrict form endpoints. If you see an error, the safest free fallback is Formspree (formspree.io). Sign up free → create a form → replace action="/website/form/crm.lead" with your Formspree endpoint URL. Everything else stays the same — styles, layout, JS validation all continue to work perfectly.

5
Phase Five Domain, SEO & Final Checks
  • 1.

    Connect your custom domain

    In Odoo, go to Website → Configuration → Settings → Domain Name. Enter your domain (e.g., innsaei.com). Then update your domain registrar's DNS: add a CNAME record pointing to your Odoo subdomain or an A record to Odoo's IP (shown on the settings page).

  • 2.

    Set page meta titles and descriptions

    In the Odoo page editor for each page, click Pages → Properties → fill in the Title and Description fields. This helps Google find your website.

  • 3.

    Check all pages look correct on mobile

    In the Odoo editor, click the phone icon to preview mobile view. All pages have been built with full mobile responsiveness — hamburger menu, stacked layouts, and proper font scaling on small screens.

  • 4.

    Publish all pages

    Each page is unpublished (draft) by default. In the editor, toggle the Unpublished → Published switch (top-right) for each page. You must publish all 7 pages to make them live.

✓ You're done! Your innsæi website is now live on Odoo. Every contact form submission will appear as a CRM Lead in your Odoo backend, where you can manage, follow up, and communicate with prospects directly using Odoo's built-in messaging system.
